PSAppDeployToolkit v4 Function Mapping

Use this reference to translate frequently used PSAppDeployToolkit 3.x helper functions to their 4.x equivalents.

The mappings below are the same values used by the Convert-LegacyCommand helper that ships with the Cloudcook PSADT Helper PowerShell module. Legacy commands on the left are rewritten to the modern names shown on the right.

3.x Function 4.x Function
Execute-MSI Start-ADTMsiProcess
Execute-MSP Start-ADTMspProcess
Execute-Process Start-ADTProcess
Show-InstallationWelcome Show-ADTInstallationWelcome
Show-InstallationPrompt Show-ADTInstallationPrompt
Show-InstallationProgress Show-ADTInstallationProgress
Show-InstallationRestartPrompt Show-ADTInstallationRestartPrompt

Tip: the PSADT Helper's Legacy Converter can also update renamed parameters when you paste older scripts into the tool.
